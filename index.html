Suggestions appear here once a chord is detected or picked.
Built around two rules of thumb: chords that share notes blend smoothly, and roots moving by a fifth feel strong. Detection works best on a clean piano triad held for a second; single notes and busy chords can fool it, so the hand-picker is there as a backup.

Phone or Mac, run it over https or localhost so the mic is allowed. If the dot won't turn on, download this file and from its folder run python3 -m http.server 8000, then open http://localhost:8000/next-chord.html.
